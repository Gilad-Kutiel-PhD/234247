בהינתן גרף מכוון ותת עץ (מושרש) שלו מסווגים את קשתות הגרף ל-4 סוגים:
\begin{enumerate}
\item
קשתות עץ
\item
קשתות קדמיות
\item
קשתות אחוריות
\item
קשתות חוצות
\end{enumerate}
\textbf{הערה:}
בגרף לא מכוון נתייחס לקשתות קדמיות וקשתות אחוריות כקשתות אחוריות.
\begin{center}
\begin{tikzpicture}[every node/.style={default node}]

	2/1
	,1/2
	,2/-1
	,3/0
	,4/1
}{
	\node(\i) at(\x,\y) {\i};
}

% EDGES
\foreach \u \v in{%
	s/1%
	,s/2%

	\draw[--] (\u) -- (\v);
}

% CROSS
\draw[blue, dashed, ->] (2) -- (1);

\end{center}
